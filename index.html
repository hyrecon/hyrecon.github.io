Projects

Neural Network-Based Channel Estimation

The lifted estimator relies on the first-order approximation ejϕ₁n ≈ 1 + jϕ₁n, which breaks down when |ϕ₂| exceeds the admissible bound — causing rapid MSE degradation. Both centered-pilot and non-centered configurations exhibit error floors outside this regime. This project explores DNN-based approaches to overcome these fundamental limitations in backscatter channel estimation.

PyTorchPythonMATLAB
Code

Phase-Aware Channel Estimation for Backscatter Links

Proposed lifted LS/LMMSE estimators that jointly resolve composite channel gain and residual phase drift in monostatic backscatter links, using pilot-based observation matrices and closed-form phase correction.

MATLABPythonSignal Processing
Demo VideoCode

Ultra-Low-Power Monostatic Backscatter Platform for Multimedia Communication

Built an end-to-end monostatic backscatter link consisting of a semi-passive tag, USRP N200 SDR reader, and a full GNU Radio signal processing chain for over-the-air multimedia (image) transmission.

GNU RadioUSRP N200PCB DesignFPGA
Code

Backscatter Range Extension via Reflection Amplifiers

Designed an active backscatter tag using a reflection amplifier with a 90° hybrid coupler and BJT gain stage, enabling range extension beyond conventional passive tag limits.

ADSAnsys HFSSVNASDR
